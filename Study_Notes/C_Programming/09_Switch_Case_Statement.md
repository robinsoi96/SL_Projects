# Switch Case Statement in C Programming

```
    switch(<expression>)
    {
        case <constant1>:
            // Statements for constant1
            break;
        case <constant2>:
            // Statements for constant2
            break;
        .
        .
        .
        default:
            // Default statements
    }
```
