# If Else Statement in C Programming

```c
    // If Condition A (Priority #1) is True, run Statement A
    if (<Condition A) { 
        <Statement A>;
    }
    // If Condition B (Priority #2) is True, run Statement B
    else if (<Condition B>) {
        <Statement B>
    }
    // Or else, just run Statement C
    else {
        <Statement C>
    }
```
